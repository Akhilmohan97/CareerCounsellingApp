## Binding Expression Cheat Sheet

```
XAML BINDING EXAMPLES FROM THIS FIX:

1. BUTTON VISIBILITY
   IsVisible="{Binding !IsGeneratingAI}"
   - Shows when NOT generating (! = NOT operator)

2. BUTTON ENABLED
   IsEnabled="{Binding !IsGeneratingAI}"
   - Enables when NOT generating

3. SECTION VISIBILITY
   IsVisible="{Binding HasAIInterpretation}"
   - Shows when interpretation exists

4. LIST BINDING
   ItemsSource="{Binding AIInterpretation.Strengths}"
   - Binds to list of strings

5. TEXT BINDING
   Text="{Binding GenerationMessage}"
   - Shows status/error message

6. TEXT WITH CONVERTER
   Text="{Binding IsGeneratingAI, StringFormat='Generating...', TargetNullValue='Generate AI Counsellor Notes'}"
   - Shows "Generating..." when true
   - Shows "Generate AI Counsellor Notes" when false

7. TEXT VISIBILITY CHECK
   IsVisible="{Binding GenerationMessage, Converter={x:Static StringConverters.IsNotNullOrEmpty}}"
   - Shows only if message is not empty
```
